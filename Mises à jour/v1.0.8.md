# 🛡️ Notes de Mise à Jour - DROPER v1.0.8 (Hotfix)

## 🛠️ Corrections & Améliorations

*   **🛡️ Correction Critique de l'Interface** : Résolution de l'erreur `ReferenceError: toast is not defined`.
    *   **Cause** : L'importation du module `toast` manquait dans le fichier d'entrée principal (`main.js`), ce qui bloquait l'affichage des notifications et l'initialisation de certains composants d'interface.
    *   **Bénéfice** : Toutes les notifications (récompenses, succès, erreurs) s'affichent désormais correctement sur tous les écrans. ✨
