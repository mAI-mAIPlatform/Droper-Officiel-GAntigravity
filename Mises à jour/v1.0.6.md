# 🛡️ Notes de Mise à Jour - DROPER v1.0.6

## 🛠️ Corrections & Améliorations

*   **🛡️ Déblocage de la Console Admin** : Résolution du bug critique empêchant l'accès au Manager même avec les bons identifiants. 
    *   **Root Cause** : Les empreintes SHA-256 dans le code étaient des valeurs de placeholder et ne correspondaient pas aux empreintes réelles calculées par les navigateurs modernes en HTTPS.
    *   **Correctif** : Mise à jour de l'authentification avec les empreintes SHA-256 réelles pour les identifiants prévus :
        *   **Login** : `admin`
        *   **Password** : `password`
        *   **Code** : `1234`
    *   **Amélioration** : Le système d'authentification a été testé pour fonctionner de manière synchrone aussi bien en environnement local (HTTP) qu'en production (HTTPS).
